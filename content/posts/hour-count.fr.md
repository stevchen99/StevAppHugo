### The Project
J'ai conçu et développé un système de Gestion des Temps et des Présences utilisé par plus de 500 employés. 

### Key Features:
* **Pointage Automatisé:** Les employés enregistrent leurs heures via une interface web ou une application mobile.
* **Backend Complexe:** Un moteur sous Node.js qui calcule les heures supplémentaires en fonction des législations du travail régionales.
* **Intégration DevOps:** Déploiement via des conteneurs Docker avec un suivi Sonar garantissant une couverture de code de 90 %.
